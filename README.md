# Featured Project — GoCars Platform

A production-grade, cloud-native marketplace platform. The product is currently paused, but the entire stack is public across the pinned repos.

```text
aws-infrastructure → aws-platform → gitops → applications

EKS · VPC · Redis      ArgoCD · ALB        Backend · Auth
IRSA · Terraform       Grafana · DNS       Meilisearch · Supabase
```

**[aws-infrastructure](https://github.com/Shinekhuu/aws-infrastructure)** — Terraform-managed AWS foundation: Amazon EKS, VPC (public/private subnets), OIDC provider, IRSA, ElastiCache Redis, Terraform Cloud remote state.

**[aws-platform](https://github.com/Shinekhuu/aws-platform)** — Shared Kubernetes platform services: ArgoCD, ArgoCD Image Updater, AWS Load Balancer Controller, ExternalDNS, Prometheus, Grafana, Loki, ACM certificates.

**[gitops](https://github.com/Shinekhuu/gitops)** — Declarative deployments for frontend, backend, auth-service, meilisearch, supabase. GitLab CI builds images, ArgoCD Image Updater promotes tags, ArgoCD continuously reconciles cluster state.

**applications** — [gocars-api](https://github.com/Shinekhuu/gocars-api) (Go backend), [auth-service](https://github.com/Shinekhuu/auth-service) (authentication microservice), [gocars-local](https://github.com/Shinekhuu/gocars-local) (local dev environment).
